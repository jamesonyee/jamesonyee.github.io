Flight Simulator

This project is a custom flight simulation experience built in Unreal Engine 5, combining real-time geospatial data with accurate flight dynamics. Using the JSBSim plugin, the simulator models realistic aircraft behavior, while Cesium integrates high-resolution 3D terrain and satellite imagery for a truly global environment.

Step into the cockpit of a fully rigged aircraft and take control using player input, which feeds directly into JSBSimâ€™s flight dynamics engine. This creates a physically plausible flying experience grounded in aeronautical principles.

Features

  • Fully rigged aircraft with animated control surfaces driven by controller input
  • Realistic flight physics powered by the JSBSim plugin
  • HUD with live data from the flight model
  • Integration with Cesium for live terrain streaming and geospatial accuracy

Key Components

Rigged Fighter Jet models

The fighter jet model, created by Yada Da Da Addda, is fully rigged to support real-time animation of control surfaces such as ailerons, rudders, and elevators. These animations are driven by controller input and synchronized with the JSBSim flight dynamics system, providing both visual feedback and physically accurate flight behavior.
This demo showcases how controller input directly affects the aircraft, with responsive control surface movement that mirrors real-world aviation mechanics. enhancing immersion.
