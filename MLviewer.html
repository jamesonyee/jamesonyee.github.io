ML2-AnchorViz

ML2-AnchorViz is a Unity-based Augmented Reality (AR) project that leverages the capabilities of Magic Leap 2 and Unityâ€™s AR Foundation. This application allows users to place four anchor points in a real-world environment and visualize 3D models within the defined plane.

Features

  • Create and manage anchor points in an AR environment.
  • Visualize a variety of 3D models in the real world.
  • Use an intuitive dropdown menu to select different models.

Requirements

  • Unity 2022.3.34f1 or later.
  • OpenXR plugin.
  • AR Foundation 5.0 or later.
  • Magic Leap SDK.
  • Magic Leap 2 device.

Getting Started

  1. Clone the repository: 
    git clone https://github.com/username/ML2-AnchorViz
  2. Import the project into Unity:
    1. Open Unity Hub and click Open.
    2. Select the cloned repository folder and let Unity import the assets.
  3. Add your own models:
    1. Import your 3D model into Unity (e.g., .fbx, .obj).
    2. Create a prefab and add your model to it.
    3. Link the prefab to the Model Prefabs array in the Spatial Anchors Test script.
    4. Update the dropdown menu options to include your model.
  4. Run the application:
    • Build and deploy the project to your Magic Leap 2 device.
    • Use the dropdown menu to select and place models in AR space.
